## Projects worth your time

### 🧠 [Text-to-SQL Fine-Tuned LLM](https://github.com/asanepranav/tinyllama-text2sql)
Fine-tuned TinyLlama-1.1B on 7,000 Spider SQL examples using 4-bit quantization + LoRA. Trained only 0.5% of parameters, cut GPU memory 10x. BLEU 32.46 on Spider validation. Live Streamlit app + weights on HuggingFace Hub.

`LoRA` `PEFT` `4-bit Quantization` `HuggingFace` `Streamlit` `TinyLlama`

### 🤖 [Agentic EDA Pipeline](https://github.com/asanepranav/agentic-eda-pipeline)
LangGraph StateGraph with 5 specialist nodes — profiler → stats → viz → insights → compiled report. Turns hours of manual EDA into minutes. Context-aware follow-up chat built in.

`LangGraph` `Groq` `LLaMA 3.3` `Pandas` `Matplotlib`

### 🔬 [Multi-Agent Research System](https://github.com/asanepranav/multi-agent-research-system)
Supervisor + 3 specialist agents (search, analyst, writer) producing full research reports from a single query. Fully local with Ollama — zero API cost. Live per-node execution trace in Streamlit UI.

`LangGraph` `Ollama` `Tavily` `Streamlit` `llama3.2`

### 📊 [Dynamic Pricing ML Engine](https://github.com/asanepranav/dynamic-pricing-model)
XGBoost + LightGBM on 100K+ real Olist orders. 8 engineered features, SHAP explainability, 34% RMSE improvement over baseline. Revenue impact simulation included.

`XGBoost` `LightGBM` `SHAP` `Feature Engineering` `scikit-learn`
